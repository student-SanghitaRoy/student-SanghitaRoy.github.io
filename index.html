Projects

2026Deep Learning · Medical Imaging · XAI

NeuraScan AI: Explainable Brain Tumour Detection from MRI

End-to-end deep learning system for multiclass brain tumour classification from MRI images. Ensemble learning across VGG16, ResNet50, and EfficientNetB0, with Grad-CAM heatmaps for visual explainability. Deployed as an interactive web app on Hugging Face Spaces.

PythonTensorFlowKerasVGG16ResNet50EfficientNetB0Grad-CAMFlaskHugging Face
2026Flask · SHAP · Business Analytics

Customer Churn Prediction & Business Analytics Dashboard

ML system predicting high-risk vs low-risk customers on the Telco dataset. Flask web app with CSV upload, real-time predictions, and business metrics — revenue at risk, retention budget, profit calculation. SHAP for explainability and Chart.js for an interactive financial dashboard.

PythonScikit-learnFlaskSHAPChart.jsXAI
2025NLP · Recommender System

Movie Recommendation System

Content-based recommendation engine using cosine similarity on TMDB metadata. Full pipeline from data preprocessing and feature engineering to a Flask web app with OMDb API for real-time poster retrieval. Deployed on a cloud platform.

PythonPandasNumPyScikit-learnFlaskOMDb API
2025Healthcare · Classification

NHANES Age Group Classification

Binary classifier distinguishing Adult (<65) vs Senior (≥65) from NHANES physiological data. Domain-driven feature engineering — HOMA-IR, glucose/insulin risk flags, BMI categories — with a full scikit-learn pipeline, 5-fold Stratified CV, ROC-AUC evaluation.

PythonScikit-learnFeature EngineeringLogistic RegressionROC-AUC
