NeuraScan AI — Brain Tumour Detection System

End-to-end deep learning system for multiclass brain tumour classification from MRI images. Deployed as an interactive web app on Hugging Face Spaces with professional documentation.

Movie Recommendation System

Content-based recommendation engine using cosine similarity on TMDB movie metadata. End-to-end pipeline: data preprocessing → feature engineering → Flask web app with OMDb API integration for real-time poster retrieval. Deployed on a cloud platform.

PythonPandasNumPyScikit-learnFlaskOMDb API
2025Healthcare · Classification

NHANES Age Group Classification

Binary classifier distinguishing Adult (<65) vs Senior (≥65) from NHANES physiological and lifestyle data. Domain-driven feature engineering — HOMA-IR, glucose/insulin risk flags, BMI categories — with a full scikit-learn pipeline, 5-fold Stratified CV, and evaluation via ROC-AUC and learning curves.

PythonScikit-learnFeature EngineeringLogistic RegressionROC-AUC
