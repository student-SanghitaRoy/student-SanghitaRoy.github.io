Projects

2025 Deep Learning · Medical Imaging · XAI

NeuraScan AI — Brain Tumour Detection System

End-to-end deep learning system for multiclass brain tumour classification from MRI images. Ensemble learning across VGG16, ResNet50, and EfficientNetB0 architectures, with Grad-CAM heatmaps for visual explainability. Deployed as an interactive web app on Hugging Face Spaces with professional documentation.

Python TensorFlow Keras VGG16 ResNet50 EfficientNetB0 Grad-CAM Flask Hugging Face Spaces
2025 NLP · Recommender

Movie Recommendation System

Content-based recommendation engine using cosine similarity on TMDB movie metadata. End-to-end pipeline: data preprocessing → feature engineering → Flask web app with OMDb API integration for real-time poster retrieval. Deployed on a cloud platform.

Python Pandas NumPy Scikit-learn Flask OMDb API
2025 Healthcare · Classification

NHANES Age Group Classification

Binary classifier distinguishing Adult (<65) vs Senior (≥65) from NHANES physiological and lifestyle data. Domain-driven feature engineering — HOMA-IR, glucose/insulin risk flags, BMI categories — fed into a full scikit-learn pipeline with 5-fold Stratified CV. Evaluated with ROC-AUC, confusion matrix, and learning curves.

Python Scikit-learn Feature Engineering Logistic Regression ROC-AUC
2026 Flask · SHAP · Business Analytics

Customer Churn Prediction & Business Analytics Dashboard

ML system that predicts high-risk vs low-risk customers on the Telco dataset using optimised probability thresholds. Built a Flask web app with CSV upload, real-time predictions, and live business metrics — revenue at risk, retention budget, profit calculation. Integrated SHAP for model explainability and Chart.js for an interactive financial impact dashboard.

Python Scikit-learn Flask SHAP Chart.js XAI
