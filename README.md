# 🌐 Sanghita Roy — Portfolio Website

Personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just one file.

**Live site:** [student-SanghitaRoy.github.io](https://student-SanghitaRoy.github.io)

---

## ✨ Features

- Animated neural network canvas in the hero section
- Typewriter effect cycling through roles
- Scroll-triggered reveal animations
- Glassmorphic sticky navigation
- Fully responsive layout
- Projects, Skills, Education & Contact sections

## 🛠️ Built with

HTML · CSS · Vanilla JavaScript

## 📁 Structure

```
index.html   ← entire site in one file
```

## 🚀 Deployment

Hosted on **GitHub Pages** — auto-deploys on every commit to `main`.
